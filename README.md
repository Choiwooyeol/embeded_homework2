## Function 

1. 소스코드에 비밀번호 네자리 0101가 지정되어있음
2. 버튼이 세개 있으며 각각 0,1,Clear 입력기능을 함
3. 버튼을 조합하여 설정된 비밀번호와 같게 입력하면 도어락이 해제됨 (서브모터가 움직임)
4. Clear버튼은 비밀번호를 잘못입력하거나 다시 도어락을 잠구고싶을때 사용
5. 도어락이 해제될시 부저가 울림
